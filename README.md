# 🎓 Student Record Management System

A simple web-based Student Record Management System developed using PHP and MySQL. This system allows users to manage student data efficiently through CRUD operations.

---

## 📌 Features

- ➕ Add new student
- 📋 View student list
- ✏️ Update student information
- ❌ Delete student record
- 🔍 Search and pagination (DataTables)

---

## 🛠️ Technologies Used

- Frontend:
  - HTML
  - CSS
  - JavaScript
  - Bootstrap

- Backend:
  - PHP

- Database:
  - MySQL

- Tools:
  - XAMPP
  - VS Code
